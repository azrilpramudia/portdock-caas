# Portdock CaaS (Container-as-a-Service)

An automated cloud hosting orchestration platform designed to streamline Docker containermanagement and deployment through an intuitive user interface.

## Tech Stack & Infrastructure
- **Frontend:** Next.js (TypeScript) & Tailwind CSS
- **Backend Core:** NestJS & Node.js
- **Package Manager:** Bun
- **Containerization:** Docker Engine API & Docker Containers
- **Target OS:** Linux Ubuntu Server (VPS)
